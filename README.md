📖 Font Arab Collection

Koleksi font Arab berkualitas untuk kebutuhan Mushaf, desain Islami, dan penulisan Arab.

---

✨ Deskripsi

Repository ini berisi berbagai font Arab seperti Uthmani, Naskh, Lateef, dan lainnya. Cocok untuk:

- Penulisan Al-Qur'an / Mushaf
- Desain Islami
- Kaligrafi digital
- Dokumen Arab

---

📂 Daftar File

- KFGQPC-Uthmanic Script
- LPMQ IsepMisbah
- Lateef
- Noto Naskh Arabic
- Traditional Arabic
- Naskh Regular

---

⚙️ Cara Menggunakan

📱 Android

1. Download file
2. Extract dengan ZArchiver
3. Install ke aplikasi (PixelLab, dll)

💻 Windows

1. Klik kanan file font
2. Pilih Install
3. Selesai
